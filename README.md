# Karel-World-Cup-2026


Karel's 2026 World Cup Strategy Engine
By Nathalie Serwaa Enam Adu-Gyamfi
Stanford Code in Place 2026 Final Project
Hello! This is my final project for the Stanford Code in Place course.
I am from Ghana, and if you know anything about us, you know we love our football. With the 2026 World Cup coming up, I wanted to create something that combines my new coding skills with the excitement of the Black Stars playing on the world stage.
What is this project?
This is a simulation game. You don't just watch the scores; you actually get to be the manager. You can choose any country (I usually pick Ghana!), and then you decide how they play. Should they attack? Should they defend? Your choices actually change the chances of winning.
I also brought in Karel the Robot from our first week of class. Karel acts as a tactical assistant, giving you updates and beeping when someone scores a goal.
How it works
I used Python to handle all 48 teams that will be in the real 2026 tournament. The program follows the real rules of the World Cup, including the group stages and the knockout rounds. I spent a lot of time making sure the math feels fair based on how good the teams are in real life.
How to run it
Make sure you have Python on your computer.
Download my file called karel_world_cup.py.
Run it in your code editor like VS Code.
A window will pop up—just follow the instructions and lead your team to the trophy!
